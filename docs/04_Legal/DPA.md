# Data Processing Agreement (DPA)

Versão: 1.0  
Última atualização: 2026-02-06  

## 1. Partes
Cliente: Controlador  
InsightAI: Operador de dados

## 2. Objeto
Tratamento de dados pessoais estritamente necessário para execução da plataforma.

## 3. Obrigações do Operador
- Processar dados apenas sob instruções do Cliente
- Garantir confidencialidade
- Implementar medidas técnicas e organizacionais

## 4. Subprocessadores
Subprocessadores são listados e auditados.

## 5. Incidentes
Notificação em prazo razoável após detecção.

## 6. Retenção e Exclusão
Configuração definida pelo Cliente.

## 7. Auditoria
O Cliente pode solicitar evidências de conformidade.

## 8. Transferência Internacional
Realizada apenas quando necessário e com salvaguardas adequadas.
