# **App Name**: InsightAI

## Core Features:

- Informational Homepage: A public-facing homepage providing key information about InsightAI, including a problem statement, explanation of functionality, benefits, and security features.
- Platform Explanation Page: A page detailing InsightAI features such as document-based chat, specialized agents, intelligent workflows, and audit/governance tools.
- Security & Compliance Page: Dedicated page that informs customers on platform security features
- Pricing Page: A clear and concise pricing page outlining the Starter, Business, and Enterprise plans with a prominent 'Falar com Especialista' (Speak to an Expert) button.
- Contact Page: A page for users to get in touch with InsightAI team
- Customer Login: A secure customer login page with email and password fields, an 'Enter' button, and a 'Forgot Password' link, designed for future authentication integration.
- AI Tool suggestion for pricing tiers: A tool to guide a client towards which of the Starter, Business and Enterprise Plans meets their requirements best.

## Style Guidelines:

- Primary color: Dark blue (#243c5a), reflecting trust and enterprise professionalism, aligned with the requested color palette.
- Background color: Very light blue-gray (#f0f4f8), a desaturated version of the primary color, for a clean, modern backdrop.
- Accent color: Cyan (#50d7ff) to highlight key interactive elements, embodying technology and innovation, matching the provided logo.
- Body and headline font: 'Inter' sans-serif, providing a clean and professional look.
- Use simple, consistent line icons to represent key concepts on the site. Prefer icons from a single consistent set.
- Employ a modular layout with a clear grid structure to ensure scalability and readability. The layout is adaptable to desktop and mobile.
- Implement subtle transitions and hover effects for an enhanced user experience. Any animation should maintain a sophisticated enterprise aesthetic.
